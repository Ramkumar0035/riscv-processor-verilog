# 32-bit 5-Stage Pipelined RISC-V Processor

## Overview
This project implements a 32-bit RISC-V processor in Verilog with a 5-stage pipeline architecture to improve instruction throughput and execution efficiency.

## Architecture
- IF (Instruction Fetch)
- ID (Instruction Decode)
- EX (Execute)
- MEM (Memory Access)
- WB (Write Back)

## Features
- 32-bit instruction support
- Pipelined architecture for improved performance
- Separate instruction and data memory
- Modular RTL design

## Tools Used
- Verilog HDL
- Xilinx Vivado / ModelSim

## Folder Structure
- src/ → Verilog source files
- testbench/ → simulation testbench
- results/ → waveform outputs
- docs/ → architecture diagrams

## Results
- Successfully simulated pipeline execution
- Verified multiple instruction test cases
